### Features of burp

#### Proxy
* Intercept web-request(like GET & POST) between the user and the web-application
* modify the requests
#### Repeater
* Send request multiple times
* Used to allow attackers to craft payloads through trial and error
* example: SQL injection
#### Intruder
* Allows brute-forcing an application
* attacker can select parameters, inject a words list and start the attack
#### Decoder
* decode captured info or encode payloads vefore sending the to target
#### Comparer
* Allows comparing 2 requests/responces easy and quickly
* allows to compare in word-level & byte-level
#### Sequencer
* allows sending ramdom tokens(example session cookies)
